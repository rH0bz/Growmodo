Design language

Dark · purple-accented · confident

Dark theme by default — not a dark-mode variant. Every surface is near-black (#141414 / #1a1a1a), text is white or muted grey (#999999), hairlines are #262626, lifted by one bold purple accent #703bf7 on every primary CTA.

Urbanist (Medium / SemiBold / Bold) is the sole typeface — a real, free Google Font, no substitution needed. Rounded cards (12px) and buttons (10px), 100px pill icon-rings, and a recurring "outer glow" box-shadow device (0 0 0 10px rgba(25,25,25,1)) framing emphasis panels — the hero feature strip, client cards, and the archive search bar.

The group's wrinkle: a property custom post type — the Properties Archive (5-facet server-side filter) and Property Single (full field schema: price, beds/baths, area, build year, key features, fee breakdown) are CPT templates, not standalone pages. Content width is a measured 1596px on the 1920 canvas.

Provenance

This catalog is generated from the Source-of-Truth at AI Website Architecture Wiki > Sites/Growmodo (decomposed from raw/Real Estate - Gowmodo/ — a 6-page, folder-per-page Figma/Anima coordinate export; the raw folder keeps the export's "Gowmodo" typo). It visualizes and does not add to the SOT.

Representative copy, not client fact: "Estatein" is the placeholder brand name baked into the export's sample copy — not a confirmed final brand name. The property cards, prices, gallery thumbnails and fee tables shown here are representative placeholders: in the real build those blocks are live WP_Query output over the property custom post type (native register_post_meta + hand-built meta box, no plugin dependency — this codebase's first CPT precedent).

Urbanist is the site's real typeface (a free Google Font — no substitution needed, unlike the prior build's licensed faces). Rendered in Growmodo's own palette/type via Tailwind Play CDN + Google Fonts (degrades to system fonts offline); icons and photos are the companion theme's real assets, referenced from Growmodo/ (26 blocks, 8 templates).
